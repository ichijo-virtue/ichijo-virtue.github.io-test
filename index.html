<ul>

<ul> (Unordered List)は、順序がない箇条書きのリストを表示する際に使用します。

1. 箇条書きリストの作成

以下は、<ul>タグを使い、果物のリストを作成する例です。

<ul>
 <li>りんご</li>
 <li>バナナ</li>
 <li>みかん</li>
</ul>
  • りんご
  • バナナ
  • みかん

<ul>タグで囲まれた部分が、リスト全体を表します。
<ul>タグの中に <li>(List Item)タグを使い、リストの項目を作成します。

<ol>

ol (Ordered List)：順序付きのリストを作成するためのHTMLタグです。
例えば、順位を表すランキングや手順の列挙など、順序が重要な意味を持つ情報をリストアップする際に使用します。

<ol>タグで囲まれた部分がリストになり、<li>タグでリストの項目を定義します。
初期設定では数字が自動的に振られますが、type属性を指定することで、ローマ数字やアルファベットなどに変更することができます。

例1：番号付きリストを作成する場合

<h2>好きな食べ物ランキング</h2>
<ol>
 <li>ハンバーグ</li>
 <li>カレーライス</li>
 <li>ケーキ</li>
</ol>

好きな食べ物ランキング

  1. ハンバーグ
  2. カレーライス
  3. ケーキ

上記のコードは、<ol>タグで囲まれた部分がリストになり、各項目を<li>タグで定義しています。
数字が自動的に振られるため、順位を表すランキングなどに適しています。

例2: アルファベットでリストアップする場合

<ol type="a">
 <li>あか</li>
 <li>べに</li>
 <li>くれない</li>
</ol>
  1. あか
  2. べに
  3. くれない

type="a"という属性を指定することで、アルファベットでリストアップすることができます。
他にも、iやIなどを指定することで、ローマ数字でリストアップすることもできます。

<a>

例1: 外部サイトへのリンクの作成

下記のようにURLを指定すると、外部サイトへのリンクを作成することができます。（実際にクリックするとジャンプします。）

<img>

画像のパス(場所)をsrc属性に指定して、その画像を表示させます。 また、下記用途のためにalt属性に代替テキストを指定できます。

  1. 画像が表示が失敗した場合の代替テキストの表示
  2. 画面の代わりの音声の読み上げ（スクリーンリーダー）

<img>タグは下記のように記述します。

<img src=`画像のパス` alt=`代替テキスト`>

例1: 画像の表示

下記の例は、HTML5.pngのパスとしてURLを指定し、画像を表示させています。

<img src="./image/HTML5.png" alt="HTML5" style="width: 64px;">

例2: 代替テキストの表示

画像の表示が失敗した場合に、alt属性に指定したテキストが表示される例です。

<img src="kakuu.png" alt="架空の画像" style ="width: 64px;">

color

テキストの色を指定します。

  • キーワード
    • 色の名前を指定することができます。例えば、red、blue、greenなどがあります。
  • RGB値
    • 赤、緑、青の3つの値を指定することができます。
      例えば、rgb(255, 0, 0)は赤色、rgb(0, 255, 0)は緑色、rgb(0, 0, 255)は青色です。
  • カラーコード
    • 16進数で表現されたカラーコードを指定することができます。
      例えば、#FF0000は赤色、#00FF00は緑色、#0000FFは青色です。

カラーコードは、以下のように省略することができます。

  • 6桁の場合、それぞれの桁が同じ場合に省略可能です。例えば、#ffffffは#fffと省略できます。
  • 3桁の場合、それぞれの桁が同じ場合に省略可能です。例えば、#112233は#123と省略できます。
  • ただし、以下の場合は省略できません。
    • 6桁の場合、それぞれの桁が異なる場合。
    • 3桁の場合、それぞれの桁が異なる場合。
/* 省略推奨 */
color: #fff; /* #ffffff */
color: #123; /* #112233 */

/* 省略できないパターン */
color: #fffff0; (6桁揃っていない)
color: #121212; (R,G,Bそれぞれが揃っていない)
color: #112234; (R,GはOKだがBが揃っていない)

/* カラー名 */ color: blue;

font-size

font-sizeは、文字の大きさを指定するプロパティです。
単位には、ピクセル(px)のように固定の数値を使用する絶対値と、
親要素やルート要素の文字の大きさに合わせて倍率を指定する相対値があります。

使用できる単位

  • px：他の要素の影響を受けず、指定したピクセル数で文字の大きさを決定します。
  • %：親要素の文字の大きさに対して、指定したパーセンテージで文字の大きさを決定します。
  • em：親要素の文字の大きさに対して、指定した倍率で文字の大きさを決定します。
  • rem：ルート要素（<html>タグ）の文字の大きさに対して、指定した倍率で文字の大きさを決定します。
p {
 font-size: 16px; /* 絶対値 */
}

例1: 文字サイズを絶対値で指定

<p class="font-size">文字の大きさを16pxに指定します。</p>

<style>
.font-size {
 font-size: 16px;
}
</style>

文字の大きさを16pxに指定します。

このコードでは、font-sizeをpxで指定しています。 絶対値のため、画面の高さや幅が変わっても文字の大きさは変化しません。

2: 文字サイズを相対値で指定

<p class="font-size">文字の大きさを2remに指定します。</p>

<style>
html {
 font-size: 16px;
}

.font-size {
 font-size: 2rem;
}
</style>

文字の大きさを2remに指定します。

このコードでは、<html>タグのfont-size: 16px;を基準として、
.font-sizeの文字の大きさを2倍に指定しています。
そのため、文字の大きさは32px（16pxの2倍）となります。

<!DOCTYPE html>

<!DOCTYPE html>は、ブラウザにHTML文書であることを伝えるための宣言です。
<!DOCTYPE>宣言をしていないとHTMLの仕様がブラウザに正確に伝わらず、意図しない表示になる可能性があります。
HTML文書の先頭には、決まり文句として必ず記述するようにしましょう。
HTML4など、過去のHTMLバージョンでは特有の<!DOCTYPE>宣言が必要でしたが、
最新のHTML5では<!DOCTYPE html>の1種類のみとなっています。

例1: HTML文書の宣言をする

<!-- 必ず先頭で宣言する -->
 <!DOCTYPE html>
<html>
 <head>
  <title>My Page</title>
 </head>
 <body>
 </body>
</html>

<html>

<html>タグは、HTML文書であることを宣言する、すべてのタグの基点となるタグです。
また、<html>タグではlang属性を使って文書の言語を指定できます。

DOCTYPE宣言の直下に記載し、文書全体を<html>タグで囲んで使用します。

<!DOCTYPE html>
<!-- 日本語で表記することを宣言する -->
<html lang="ja">
 <head>
  <title>htmlタグとは</title>
 </head>
 <body>
  <h1>htmlタグの使い方</h1>
 </body>
</html>

Webページを作るときには、決まり文句として上記のように、htmlタグを書くようにしましょう。

<head>

<head>タグは、ページの情報、タイトル、CSSなどの外部ファイルを読み込むリンクなど、ページに関する情報を記述する場所です。 <head>タグ内の情報はページを見るユーザーに直接表示されることはありませんが、検索エンジンやブラウザに伝わるため、 ページを公開する上で重要な役割を持っています。

例1. 基本的な情報を記述する

<head>
 <meta charset="UTF-8" />
 <meta name="viewport" content="width=device-width,initial-scale=1" />
 <title>My Page</title>
 <link rel="stylesheet" href="css/style.css" />
</head>
  • charset：ページで使用される文字の種類（文字コード）を指定します。
  • viewport：ページをレスポンシブデザイン（スマートフォンやタブレットで見やすくする）に対応させる指定です。
  • <title>：ページタイトルの指定です。
  • <link>：スタイルシート（CSSファイル）などを読み込みます。href属性には、適用したいファイルパスを指定します。

<title>

 head タグ内に記載し、Webサイトやページにタイトルを定義するためのタグです。
定義したタイトルは検索結果に表示されるだけでなく、ブックマークやSNSでURLを共有した際のタイトルとしても表示されます。
ページ内容が伝わるシンプルなタイトルにすることは、SEO対策にも有効です。

例1：タイトルを定義します

<head>
 <title>TOKYO BURGER | 東京 こだわりの鉄板焼きハンバーガー</title>
</head>

<meta>

 head タグ内の子要素として記載します。
 meta タグを記載することで、Webページの説明や文字コードなどの情報を設定できます。

 meta タグで設定できる情報の例

  • 文字コードの指定
  • Webサイトの説明文の設定
  • 各種ブラウザ(デバイス)でのサイズ最適化
  • コンテンツの種類

例1: Webページの情報を指定する

<head>
 <!-- 文字エンコーディング(文字コード)の指定 -->
 <meta charset="UTF-8" />
 <!-- Webサイトの説明文。検索エンジンの検索結果にも表示されるため重要 -->
 <meta name="description" content="..." />
 <!-- 各種ブラウザでサイズを最適化する -->
 <meta name="viewport" content="width=device-width,initial-scale=1.0" />
</head>

<meta charset="utf-8" />

<meta charset="utf-8" />は、Webページで使用する文字コードを指定するタグです。
HTMLファイルの<head>タグ内で指定し、ブラウザに対してHTMLファイルがUTF-8で記述されていることを伝えます。

文字コードとは

文字コードとは、コンピュータが文字を扱うために文字に割り当てる固有の番号のことです。
設定すると、文字や記号をコンピュータが理解できるようになります。

日本語のWebページの場合、世界中のほとんどの言語をサポートしているという理由から、UTF-8が一般的に使われています。

HTMLファイルの文字コードと、<meta charset="utf-8" />でブラウザに伝える文字コードを
一致させることで文字化け防止になるため、必ず設定するようにしましょう。

例1: Webページの文字化け防止

<head>
<!-- 文字コードを指定 -->
 <meta charset="UTF-8" />
 <meta name="viewport" content="width=device-width,initial-scale=1" />
 <title>My Page</title>
 <link rel="stylesheet" href="css/style.css" />
</head>

このコードでは、<head>タグ内で<meta charset="utf-8" />を指定することにより、
ブラウザにHTMLファイルがUTF-8で記述されていることを伝えています。

<link>

<link>は、htmlファイルと外部ファイル（スタイルシートやファビコンなど）を関連づけるタグです。
<link>タグは、基本的に<head>タグ内に記述します。

rel属性・href属性とは

  • rel属性：relation（関係）の略で、タグを設置するページと関連づけるファイルとの関係性を示します。
  • href属性：関連づけるファイルのURLを記述します。

例1.スタイルシートを関連づける

<head>
 <link rel="stylesheet" href="style.css" />
</head>

このコードでは、スタイルシートを<link>タグで関連づけています。

  • rel属性：stylesheet
  • href属性：cssファイルのURL

例2: ファビコンを指定する

<link rel="icon" href="img/favicon.ico">

このコードでは、ブラウザのタブやブックマークに表示するアイコン画像（ファビコン）を<link>タグで関連づけています。

  • rel属性：icon
  • href属性：アイコン画像のURL

<span>テキスト</span>

この例では、<span>タグで囲まれたテキスト「りんご」の色が赤に変わります。

font-family

CSSのfont-familyは、Webページ上で表示するテキストの書体（フォント）を指定するプロパティです。
Webページを作成する際に、テキストのフォントの種類(見た目)を指定することができます。

body {
 font-family: "メイリオ", sans-serif;
}

例1 : 特定のフォントを指定する

/* 'font-family: 'フォント名', 代替フォント名; */
font-family: 'Noto Sans JP', sans-serif;

上記のコードでは、スペースが含まれるフォント名を指定する場面なので、指定したいフォント名を引用符(''など)で囲むことが必須になります。

代替フォント名は、指定したフォントが存在しない場合、または表示できない場合に、代わりに表示されます。
フォントが存在しない理由として、フォントがインストールされていない場合や、
サポートされていないフォント形式である場合などが考えられます

例2 : 複数のフォントを指定する

/* font-family: フォント名1, フォント名2, 代替フォント名; */
font-family: Arial, Helvetica, sans-serif;

上記のコードのように、複数のフォント名をカンマ区切りで指定することができます。
左から順番に、Webブラウザが表示可能なフォントかどうかを探し、最初に表示可能だと判断されたフォントが使用されます。
全てのフォントが表示できない場合には、代替フォントが使用されます。

letter-spacing

文字間隔を指定します。
主に、数値+pxまたは数値+emで指定します。
相対値であるemで記述すれば、文字のサイズを変更してもそれに応じたスペースが設定されます。
推奨の値は0.05〜0.1em(5%〜10%)。

例1：emを使用して文字間隔を指定します。

p {
font-size: 20px;
letter-spacing: 0.05em; /* 文字のサイズに対して5%分のスペースを空ける */
}

0.05emは文字サイズの5%となります。
letter-spacingでは、相対的な比率で指定したほうが文字のバランスが良くなる傾向にあるため、em指定を推奨します。

line-height

行の高さを指定します。
単位を使わず、数値のみで指定することが推奨されています。
1.5～2が一般的に見やすい数値とされています。

例1：行の高さを文字サイズの1.5倍に設定します

文字サイズが20pxのテキストにline-height: 1.5;を指定すると、行の高さは30pxとなります。

<style>
.test {
 font-size: 20px;
 line-height: 1.5;
}
.test p {
 font-size: 10px; /* 10px × 1.5 = 高さ15pxになる*/
}
</style>
<div class="test">
 <p>line-heightを使うと、</p>
 <p>行間がいい感じに広がります。</p>
</div>

line-heightを使うと、

行間がいい感じに広がります。

「数値+px」「数値+em」などで指定することもできますが、em等の相対値で指定すると、文字のサイズに関わらず、
親要素で計算されたline-heightが継承されてしまいます。
「1.5」など、単位を使わずに数値のみでの指定すると、Webページの行間が統一されるため推奨されています。

box-shadow

box-shadowは、ブロック要素の周りに影を追加するプロパティです。
値は半角スペースで区切り、下記の順番で指定します。

  • ① 横の位置
  • ② 縦の位置
  • ③ ぼかし
  • ④ 広がり
  • ⑤ 色
  • ⑥ inset（指定すると影が内側に追加されます）

①〜②の設定は必須で、③〜⑥の設定は任意です。
正の値を指定すると、右と下に影がつきます。反対に、マイナス値の場合、左と上に影がつきます。

例1. 右下に影を追加する

<div class="block"></div>

<style>
.block {
 /* 横と縦に5px,ぼかし20px,広がり-5pxの影を追加する */
 box-shadow: 5px 5px 20px -5px #cdcdcd;
 width: 100px;
 height: 100px;
 border-radius: 10px;
}
</style>

例2. 左下に影を追加する

<div class="block"></div>

<style>
.block {
 /* 横に-5px,縦に5px,ぼかし20px,広がり-5pxの影を追加する */
 box-shadow: -5px 5px 20px -5px #cdcdcd;
 width: 100px;
 height: 100px;
 border-radius: 10px;
}
</style>

このコードでは、box-shadowで横の向きを-5pxとマイナス値で指定し、影を左側に追加しています。

例3. 内側に影を追加する

<div class="block"></div>

<style>
.block {
 /* 横・縦に0px,ぼかし20px,広がり-5px,内側の影を追加する */
 box-shadow: 0px 0px 20px -5px #cdcdcd inset;
 width: 100px;
 height: 100px;
 border-radius: 10px;
}
</style>

このコードでは、box-shadowでinsetを指定し、内側に影を追加しています。

border-radius

border-radiusは、要素の角を丸くするために使用します。例えば、角が鋭利な四角形を丸くすることができます。

使用方法としては、各角に半径を指定することで、任意の大きさの丸みをつけることができます。

/* 四隅に10pxの丸みを指定 */

border-radius: 10px 20px;

/* 左上が10px、右上と左下が20px、右下が30pxの丸みを指定 */

/* 左上、右上、右下、左下の順に丸みを指定 */

list-style

<ul>,<ol>,<li>の箇条書きリストのスタイルを一括指定できます。
値は半角スペースで区切って記述するか、個々に記述したい場合は次のプロパティを使用します。

  • list-style-type：リストの先頭についているマーカーの見た目を変更する際に使用します。
    • disc：黒丸（ulとliの初期値）
    • decimal：数字（olの初期値）
    • circle：中空円
    • none：表示しない
  • list-style-image：マーカーを画像で指定します。url(画像パス）で記述します。

display

CSSのdisplayプロパティは、HTML要素の表示方法を指定するために使用されます。
displayプロパティには、要素を非表示にするための値や、
要素をブロックレベル要素やインライン要素に変更するための値などがあります。

display: block;

要素をブロック要素（ブロックレベル要素）に指定します。

  • 前後改行が入り、縦に要素が並びます。
  • 幅と高さを指定できます。
  • marginとpaddingで自由に余白を調節できます。
  • 要素の幅を指定しない限り、親要素の100%となるため、要素の配置は指定できません。

初期値がブロック要素のタグ

  • <div>、<section>、<p>、<ul>, <li>など

display: inline;

要素をインライン要素に指定します。

  • 前後に改行が入らず、横に要素が並びます。
  • 幅と高さは指定できません。
    • <img>は例外で、幅と高さを指定することができます。
  • 上下の余白が調節できません
    • 上下のpaddingは指定可能ですが、上下のコンテンツの余白と重なって表示されるため、非推奨です。

displayの初期値がインライン要素のタグ

  • <span>、<a>、<img>など

display: inline-block;

  • inline同様、要素が横に並びます。
  • 幅と高さを指定できます。
  • marginとpaddingで余白を自由に調節できます。
  • 要素の配置を指定できます。
  • ブロック要素との使い分けは、縦並び/横並びのどちらにしたいかで考えると良いです。

display: none;

  • ブラウザ上で非表示になります。
  • レスポンシブで表示切り替えしたい時などに使用する。

opacity

CSSのopacityは、要素の透明度を設定するプロパティです。値は0から1までの小数で指定します。
0に近いほど透明度が高く、1に近いほど不透明度が高くなります。

  • 0：完全に透明
  • 1：完全に不透明

例1：要素を表示します

<div class='opacity-ex'>
 <p> 透明になります</p>
</div>

 .opacity-ex {
   background: rgba (132, 127, 127, 1);
 }
</style>

透明になります

例2：表示させた要素を透明にします

<div class='opacity-ex'>
 <p> 透明になります</p>
</div>

<style>
 .opacity-ex {
   background: rgba (132, 127, 127, 1);
   opacity: 0;
 }
/* 要素全体が透明になるので、p要素「透明になります」や背景色が見えなくなる。rgba()との違いに注意 */
</style>

透明になります

注意点

  • opacityは要素全体の透明度を指定します。そのため、要素内の背景や文字だけを薄くしたい場合は、
    個別にrgba()で記述する必要があります。
  • opacity: 0;とdisplay:none;も似ていますが、opacityは非表示でなく透明になるだけなので、 要素の領域は残ります。
    そのため、クリックなどのアクションができるといった違いがあります。

セレクタの使い方を学ぼう

要素型セレクタ

要素型セレクタは、HTML文書内の要素の種類に基づいてスタイルを適用するためのセレクタです。
要素型セレクタは、要素のタグ名を指定してスタイルを適用します。

要素名 {
 スタイルのプロパティ: スタイルの値;
}

要素型セレクタの特徴

  • 要素型セレクタは、HTML要素のタグ名を指定するため、タグ名が一致する要素全てにスタイルが適用されます。
  • 要素型セレクタは、クラス名やID名を指定するよりも汎用的なセレクタです。
  • 要素型セレクタは、CSSの優先度が最も低いため、他のセレクタによって上書きされることがあります。

例1: 特定の要素にスタイルを適用する

以下の例では、p要素内のテキストの色が赤色になります。

<p>赤色のテキスト</p>
<p>青色のテキスト</p>

<style>
p {
 color: red;
}
</style>

例2: 複数の要素にスタイルを適用する

以下の例では、h1、h2、h3要素のテキストのフォントサイズが24ピクセルになります。

<h1>見出し1</h1>
<h2>見出し2</h2>
<h3>見出し3</h3>

注意点

  • 要素型セレクタは、HTML要素のタグ名を指定するため、同じタグ名を持つ要素全てにスタイルが適用されます。
    そのため、意図しない要素にもスタイルが適用される可能性があります。
  • 要素型セレクタは、CSSの優先度が最も低いため、他のセレクタによって上書きされることがあります。

全称型セレクタ(*)

CSSの全称セレクタ（*）は、HTML文書内の全ての要素を選択するためのセレクタです。
つまり、どの要素にも共通するスタイルを一括で指定することができます。

* {
 margin: 0;
 padding: 0;
}

上記のように、全称セレクタを使用することで、全ての要素のmarginとpaddingを0に設定することができます。

例1:共通のスタイルを指定する

全ての要素に対して、共通のスタイルを指定することができます。

<style>
* {
 font-family: Arial, sans-serif;
 color: green;
}
</style>

上記のように、全ての要素に対して、フォントファミリーと文字色を指定することができます。

例2:要素のスタイルをリセットする

ブラウザによっては、要素にデフォルトで設定されているスタイルが異なる場合があります。
全称セレクタを使用することで、要素のスタイルをリセットすることができます。

* {
 margin: 0;
 padding: 0;
 border: none;
 font-size: 100%;
 font-weight: normal;
 line-height: 1;
 vertical-align: baseline;
}

上記のように、全ての要素に対して、マージン、パディング、ボーダー、フォントサイズ、
フォントウェイト、行の高さ、垂直方向の配置をリセットすることができます。

class属性に指定した文字列（class名）の先頭に . をつけることで、class名がつけられた要素にスタイルを適用するセレクタです。

class名の先頭に数字を使用することはできません。

idセレクタ

id属性に指定した文字列（id名）の先頭に # をつけることで、id名がついた要素にスタイルを適用するセレクタです。
特定の要素にスタイルを適用する場合や、ページ内リンクの指定に用いられます。

例1:特定の要素にスタイルを適用する

idセレクタを用いることで、特定の要素に対してスタイルを適用することができます。
例えば、ヘッダー部分に背景色を付ける場合は、以下のように書きます。

<header id="header"></header>

<style>
#header {
 width: 100%;
 height: 80px;
 background-color: #ccc; /* #header部分の背景色を指定 */
}
</style>

例2:ページ内リンクの指定

idセレクタは、ページ内リンクの指定にも用いられます。
例えば、以下のようなリンクがある場合、#section1は、ID属性がsection1である要素を指定しています。
これにより、リンクをクリックするとsection1の位置までスクロールするようになります。

<a href="#section1">Section 1</a>

注意点

  • id名の先頭に数字を使用することはできません。
  • HTML文書内で同じid名は1度しか指定できません。
    id属性はHTML文書内で一意である必要があり、同じid名を指定すると複数の要素が同じid属性を持つことになるため、
    文書内で一意であるという要件を満たせなくなるからです。
